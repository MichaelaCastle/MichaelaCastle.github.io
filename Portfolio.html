Ever since I was young, I loved programming. The first thing I ever got to program was an Arduino, which my father gifted to me when I was about eight. I loved creating and learning how to make new things with it. When I turned about twelve, I found that I also really liked to create stories and art. I spent hours upon hours creating a detailed world with character that I still use to this day. When you take all three interests and combine them, you get a game! I plan on turning that story mentioned before into a game one day. I am still developing the same characters I made when I was twelve into more and more realistic characters. I have also been defining more information about how magic and mutations work in my universe.

Projects

Prismoria, a world and story that I've been developing since I was 12 years old. This is a passion project that I'm still waiting on starting. This story has become too complex for me to be able to develop this on my own, and I still need to figure out how to represent it in a game. This project however is a custom webpage created to show off details of this world that have been fleshed out.

Sudoku, a recreation of the classic Sudoku game in JavaScript, HTML, and CSS. This includes extra features such as boards with more rules on top of the classic rules, a notetaking system, and error checking so the user knows when the broke a rule.

DND Database, which uses a public API to retrieve and parse data. It then displays said data to the user. The user can filter and search for items within the database.

Nucifera, an academic solo project made in Unity without utilizing Unity's built-in ridgedbody or physics system. It's a 2D Shoot-'em-up game that I made all the artwork, AI, and code for.

Good Luck~, a group project made in Monogame. I made all the graphics and I also contributed to the enemy behaviors, menu system, and player attacks.

Factory Simulator, a 6-month long project created in an attempt to learn more about databases. I did not learn anything about databases, but I got a lot of experience with Unity.
